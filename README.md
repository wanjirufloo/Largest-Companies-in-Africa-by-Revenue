# Africa Top Companies Web Scraper (Python)

## Overview
Built a Python web scraper to extract data on the largest companies in Africa by revenue from Wikipedia.

## Data Collected
- Company rank
- Company name
- Industry
- Revenue
- Headquarters location

## Processing
Cleaned and structured the scraped data into a usable dataset for analysis.

## Outcome
The dataset enables comparison of industry representation and geographic distribution of top-performing companies across Africa.
